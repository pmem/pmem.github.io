Code Samples

Learn, build, and deploy persistent memory features in your application using an extensive collection of documentation, structured tutorials, webinars, forum, and API documentation. The following are useful links to code examples across several projects.

PMDK

The base directory for the code examples for PMDK can be found at https://github.com/pmem/pmdk/tree/master/src/examples. Each directory contains code samples for each library:

  • Libpmem - https://github.com/pmem/pmdk/tree/master/src/examples/libpmem
  • Libpmem2 - https://github.com/pmem/pmdk/tree/master/src/examples/libpmem2
  • Libpmemblk - https://github.com/pmem/pmdk/tree/master/src/examples/libpmemblk
  • Libpmemlog - https://github.com/pmem/pmdk/tree/master/src/examples/libpmemlog
  • Libpmemobj++ - https://github.com/pmem/pmdk/tree/master/src/examples/libpmemobj%2B%2B
  • Libpmemobj - https://github.com/pmem/pmdk/tree/master/src/examples/libpmemobj
  • Libpmempool - https://github.com/pmem/pmdk/tree/master/src/examples/libpmempool
  • Librpmem - https://github.com/pmem/pmdk/tree/master/src/examples/librpmem
  • Pmreorder - https://github.com/pmem/pmdk/tree/master/src/examples/pmreorder
